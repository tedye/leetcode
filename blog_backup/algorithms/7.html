Question
Reverse digits of an integer. Example1: x = 123, return 321 Example2: x = -123, return -321
Analysis
    本题主要考察的就是对于corner case的处理，也就是overflow的处理。复杂度为输入integer的位数。代码如下。
Code 
class Solution(object):
    def reverse(self, x):
        """
        :type x: int
        :rtype: int
        """
        sign = (x < 0)
        x = abs(x)
        result = 0
        while x:
            result *= 10
            result += x%10
            x //= 10
        if result > 0x7fffffff and not sign or result > 0x7fffffff+1 and sign:
            return 0
        if sign:
            return -1*result
        else:
            return result
